# Online Quiz Website

This is an online quiz website where users can take exams. The project is developed using HTML, JavaScript, and CSS.

## Features
- User-friendly interface for taking quizzes
- Timer for each exam
- Score calculation at the end of the quiz

## Technologies Used
- HTML
- JavaScript
- CSS
